# Install Project
## Backend
``` cd backend ```
### Config

- copy file .env.example and rename this to .env
- Install **pipenv**: ``` pip install pipenv ```
- Install packages:
``` pipenv install```
### Install something package
``` pipenv install <name-package> ```
* example:
``` pipenv install python-dotenv ```
### Run server
``` flask run ```
## Frontend

``` cd frontend ```
### Install packages
``` npm install ```
### Run server
``` npm run start ```
