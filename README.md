# Install Project
## Backend

``` cd backend ```
### Install packages
``` pip install -r requirements.txt ```
### Run server
``` flask run ```
## Frontend


``` cd frontend ```
### Install packages
``` npm install ```
### Run server
``` npm run dev ```
